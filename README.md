# Java Projects

This repository contains beginner-friendly Java projects developed to practice core Java programming concepts.

## Projects

### 1. Guessing Number Game
A console-based game where the user tries to guess a randomly generated number within a limited number of attempts.

**Concepts Used:**
- Java Basics
- Loops
- Conditional Statements
- Random Class
- User Input (Scanner)

---

### 2. Student Grade Calculator
A Java application that calculates the total marks, average percentage, and assigns grades based on student marks.

**Concepts Used:**
- Variables
- Arrays
- Arithmetic Operations
- Conditional Statements
- Scanner Class

---

### 3. ATM Interface
A simple ATM simulation that allows users to perform basic banking operations such as:
- Check Balance
- Deposit Money
- Withdraw Money
- Exit

**Concepts Used:**
- Object-Oriented Programming (OOP)
- Classes and Objects
- Methods
- Conditional Statements
- Loops
- Scanner Class

---

## Technologies Used

- Java

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/sowmya-220803/java.git
   ```

2. Open the project in VS Code or any Java IDE.

3. Compile the Java file:
   ```bash
   javac FileName.java
   ```

4. Run the program:
   ```bash
   java FileName
   ```
